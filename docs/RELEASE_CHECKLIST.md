# Shindan OS Release Checklist

実施日：`YYYY-MM-DD`  担当：`__________`  デプロイ先：`__________`

## 1. ビルド確認

- [ ] `npm install`（または`pnpm install --frozen-lockfile`）が成功する
- [ ] `npm run lint`が成功する
- [ ] `npm run validate:questions`が成功、または既知の警告のみで終了する
- [ ] `npm run validate:backup`が成功する
- [ ] `npm run build`が成功する
- [ ] TypeScriptエラー、未使用import、未使用変数がない
- [ ] ビルド警告の内容を確認し、既知の年度依存警告以外を解消した
- [ ] private問題・教材本文・秘密情報が公開成果物に含まれていない

## 2. デプロイ確認

- [ ] VercelのFramework PresetがNext.jsになっている
- [ ] `/`がHTTPSで表示できる
- [ ] `/manifest.json`が200で返る
- [ ] `/sw.js`が200で返り、`Service-Worker-Allowed: /`が付く
- [ ] `/icon.svg`、`/icon-maskable.svg`、`/apple-touch-icon.png`が200で返る
- [ ] 通常URLではQAボタンが表示されない
- [ ] `?debug=1`の場合だけQAボタンが表示される
- [ ] QAデータ生成とリセット前に確認ダイアログが表示される

## 3. スマホ実機確認

- [ ] iPhone Safariで表示できる
- [ ] 390px前後で横スクロールが出ない
- [ ] ヘッダーと下部ナビが安全領域に収まる
- [ ] ホーム画面に追加できる
- [ ] ホーム画面起動時にstandalone表示になる
- [ ] 右上の赤い復習バッジが見える
- [ ] 10分モードとランダム出題を開始できる
- [ ] 4択と入力式問題を回答できる
- [ ] 模範解答を確認して自己採点できる
- [ ] 解説、間違えやすいポイント、試験ポイントが読める
- [ ] 不正解・ほぼ正解でWeak論点に追加される
- [ ] 復習タスクが作成される
- [ ] Review画面で未確認・要レビューを判別できる
- [ ] 証跡付き確認済み操作ができる
- [ ] データをエクスポートできる
- [ ] 正常なバックアップをインポートできる

## 4. データ保護確認

- [ ] エクスポートJSONに`appName`、`version`、`exportedAt`がある
- [ ] 回答履歴、Weak、進捗、ストリーク、証跡、QA状態が含まれる
- [ ] インポート前に上書き確認が表示される
- [ ] 不正JSONを選択してもクラッシュせずエラーになる
- [ ] 未対応バージョンのJSONが拒否される
- [ ] リセット前にバックアップ推奨と確認入力が表示される
- [ ] 「リセット」以外では削除されない
- [ ] リセット後に初期状態へ戻る
- [ ] 最終エクスポート・インポート日時が更新される

## 5. PWA・オフライン確認

- [ ] manifestのname、short_name、theme_color、display、start_urlが正しい
- [ ] ホーム画面でアイコンが表示される
- [ ] ブラウザ／status barへtheme colorが反映される
- [ ] Service Workerが登録・有効化される
- [ ] 初回オンライン表示後にページを再表示した
- [ ] 機内モードでアプリの最低限の画面が表示される
- [ ] オフラインで問題表示・回答・localStorage保存ができる
- [ ] オンライン復帰後も学習履歴が残っている

## 6. リリース判定

- [ ] 重大な未解決エラーがない
- [ ] 既知の警告・制約をREADMEまたはリリースノートへ記載した
- [ ] 公開前バックアップを保存した
- [ ] リリース可否を記録した：`GO / NO-GO`
